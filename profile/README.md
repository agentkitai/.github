# AgentKit AI

Open-source infrastructure for **trustworthy** AI agents.

> **[AgentLens](https://github.com/agentkitai/agentlens)** is our flagship — observability for AI agents with a **SHA-256 hash-chained, tamper-evident audit log** (chain verification + signed export), built for EU AI Act Article 12 record-keeping. MCP-native, ingests OpenTelemetry GenAI traces, real-time dashboard.

## Projects

| Project | Description |
|---------|-------------|
| 🔍 [AgentLens](https://github.com/agentkitai/agentlens) | **Flagship** — observability & tamper-evident audit trail for AI agents. |
| [AgentGate](https://github.com/agentkitai/agentgate) | Human-in-the-loop approval gateway + reactive guardrails. |
| [Lore](https://github.com/agentkitai/lore) | Cross-agent memory — agents publish lessons and query shared knowledge. |
| [AgentEval](https://github.com/agentkitai/agenteval) | Testing & evaluation framework for AI agents. |
| [FormBridge](https://github.com/agentkitai/formbridge) | Mixed-mode agent-human form submission. |

**Tooling:** [AgentKit CLI](https://github.com/agentkitai/agentkit-cli) · [AgentKit Stack](https://github.com/agentkitai/agentkit-stack)
