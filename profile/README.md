# AgentKit AI

Open-source infrastructure for AI agents — observability, approvals, memory, forms, and orchestration.

## Projects

| Project | Description |
|---------|-------------|
| [AgentLens](https://github.com/agentkitai/agentlens) | Observability and audit trail platform for AI agents. MCP-native, real-time dashboard. |
| [AgentGate](https://github.com/agentkitai/agentgate) | Approval workflows for AI agents. Human-in-the-loop policy enforcement. |
| [Lore](https://github.com/agentkitai/lore) | Cross-agent memory SDK. Agents publish lessons, query shared knowledge. |
| [FormBridge](https://github.com/agentkitai/formbridge) | Mixed-mode agent-human form submission infrastructure. |
| [AgentEval](https://github.com/agentkitai/agenteval) | Testing and evaluation framework for AI agents. |
| [AgentKit Mesh](https://github.com/agentkitai/agentkit-mesh) | Agent-to-agent discovery and delegation via MCP. |
| [AgentKit CLI](https://github.com/agentkitai/agentkit-cli) | CLI orchestrator for the AgentKit ecosystem. |
| [AgentKit Guardrails](https://github.com/agentkitai/agentkit-guardrails) | Proactive guardrails — auto-tighten policies when metrics breach thresholds. |
| [AgentKit Stack](https://github.com/agentkitai/agentkit-stack) | Docker Compose stack for local AgentKit development. |
